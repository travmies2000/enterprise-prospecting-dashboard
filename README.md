# Enterprise Prospecting Dashboard

The **Enterprise Prospecting Dashboard** is a system-first operational control center designed to help holiday rental hosts manage inquiries, track pipeline value, and monitor automation performance. This dashboard acts as the missing "operational layer" that sits on top of platforms like [Travelnest](https://travelnest.com/referral-link-tn?utm_campaign=referrer_submission&utm_medium=email&utm_source=braze&lid=m3xvdlz5f5qj).

## Features
* **Pipeline Visibility**: Automatically calculate and display your total potential booking revenue.
* **Smart-Filter Interface**: Effortlessly sort guests by status (Pending, Followed-up) to maintain workflow focus.
* **Automation Monitoring**: Real-time side panel displaying system status and automation logs for peace of mind.
* **Glassmorphism SaaS Design**: A professional, dark-mode aesthetic that provides high-level business insights at a glance.

## Quick Start
1. **Clone the Repository**: Download the files to your local environment.
2. **Configure Webhooks**: 
    * Update `script.js` with your Google Sheet Webhook URL (for data) and your automation platform URL (for logs).
3. **Deploy**: Push the files to your GitHub repository and enable **GitHub Pages** in your repository settings to make the dashboard live.
